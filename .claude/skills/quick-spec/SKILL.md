name: quick-spec
description: Grill the user on a feature idea, then generate a high-level spec ready for Plan Mode. Use when you have a feature idea and want a quick spec without the full PRD process.
---

I'll grill you on this feature idea to flesh out the details, then generate and save a spec ready for Plan Mode.

First, let me ask you a few focused questions about this feature:

1. **Core problem**: Why does this feature exist? What problem does it solve or what value does it add?

2. **Scope**: What are the boundaries? What's explicitly NOT included?

3. **Success**: How do you know this feature is done? What does success look like?

4. **Constraints**: Any technical, time, or dependency constraints I should know about?

Once I have these answers, I'll generate your spec in the format ready for Plan Mode and save it to `_specs/`.

Ask one question at a time, wait for your answers, then produce a spec that maps to:
- Summary
- Functional Requirements
- Possible Edge Cases
- Acceptance Criteria
- Testing Guidelines

Then ask you to confirm it before saving to `_specs/`.
